## ℹ️ What is ua-parser-modern?  

ua-parser-modern is a simple tool that helps you find out which browser and operating system you are using. It looks at a string called "User-Agent" and tells you the name and version of your browser and OS. This tool has been rewritten using modern JavaScript and TypeScript to work better and faster.  

You do not need to know programming to use this. This guide will help you download and run ua-parser-modern on your Windows PC.

This command runs an example script included with the software. It will show you browser and OS info based on some test data.

- Open the example.js file in a text editor like Notepad.  
- Find the part of the file that contains a sample User-Agent string (it looks like a long line of text with browser details).  
- Replace it with your copied User-Agent string.  
- Save the file and run `node example.js` again to see your device info.
